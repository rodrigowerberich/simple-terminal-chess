# Chess in Terminal 

The idea is to create a simple chess application

A regular chess terminal application for two players.

The game consists of 16 pieces on each side
2x Rook - RA, RB 
2x Knight - KB, KG
2x Bishop - BC, BE
1x Queen - Q
1x King - K
8x Pawn - PA,PB,PC,PD,PE,PF,PG,PH
